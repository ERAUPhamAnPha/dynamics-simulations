# Dynamics Simulations

This repository contains computational simulations related to engineering dynamics.

## Projectile Motion Simulation

This project models 2D projectile motion using:

- Newtonian kinematics
- Vector decomposition
- NumPy for computation
- Matplotlib for visualization

The simulation:
- Computes x and y positions over time
- Calculates theoretical range
- Visualizes trajectory
